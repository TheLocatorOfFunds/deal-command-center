🚫 DNC review SOP

For Eric & Inaam · when a lead shows the do-not-contact banner in Review

1. Read the banner

The banner names the flagged contact, the number, which flag (do-not-call / do-not-text), and the recorded reason. Start there — that's the "why."

2. Act by reason — then Mark Reviewed

ReasonWhat it meansDo this
Disconnected / bad numberthe number is dead — the person is not opted outdon't dial it; use another contact if there is one → if none, see step 3
Do-not-text only (call not flagged)texting off, calling is finecall as normal, just don't text
Do-not-call setdon't dial that contactuse another callable contact (heir / relative) if one exists
Genuine opt-out ("stop contacting me")legal hard-stop (TCPA)no calls or texts to them, ever — keep the lead open, outreach just stops; work it through a co-claimant / heir if one exists
Auto-detected from notesthe system flagged a phraseread the note → real opt-out = hard-stop; false positive = clear the contact's DND toggle (note why), then Mark Reviewed

3. No callable contact left

If every contact is do-not-call or disconnected: flag it "needs a new number" and leave it — do NOT clear it to ready-to-call until a good number is found. Don't mark it dead — the surplus is still recoverable; it just needs a fresh number (skip-trace).

What "Mark Reviewed" does: it only removes the lead from the Review Queue. It does not call, strip a number, or change a DND flag. So only clear it after you've handled it per the steps above.
⚖️ Compliance: a real "do not contact me" is a legal hard-stop. If you can't tell a bad number from a genuine opt-out, treat it as an opt-out and check with Nathan.
